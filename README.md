# TRON GAME

A version of the classic TRON into an arcade-like Game. 5 Levels, each with more and more walls, and ennemies that will respawn when killed. just how many do you think you can kill before you reach a wall?

like the classic tron, walls appear behind every player. but when one die, their wall disappear. this allow a new bot to join the arena.
#

## How to Play:

zqsd/arrow keys: Moving

Esc: Menu/Pause Screen
## 


### Codes:

-fast : double the speed of all

-slow : halve the speed of all

-rainbow : have the player gain a rainbow line
